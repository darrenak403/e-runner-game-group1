# 🏃 Runner Game

A dynamic endless runner game built with **Unity**. Experience fast-paced action, dodging obstacles, and collecting coins in an ever-changing environment.

## 🌟 Key Features

*   **Endless Gameplay**: Procedurally generated levels that keep the challenge fresh.
*   **Intuitive Controls**: Precision swipe controls for mobile-friendly navigation.
*   **Dynamic Environments**: Background management and item spawning for an immersive experience.
*   **Collectibles**: Gather coins to increase your score.
*   **Game Over System**: Integrated logic for handling failures and restarting.
*   **High-Quality Audio**: Immersive sound effects and background music.
*   **Premium UI**: Powered by **TextMesh Pro** for sharp, scalable text.

## 🛠️ Technical Implementation

*   **SwipeController**: Handles player movement through touchscreen or mouse gestures.
*   **LevelGenerator**: Manages the instantiation and recycling of level segments.
*   **ItemSpawner**: Distributes obstacles and collectibles throughout the path.
*   **BGManager**: Synchronizes background elements with player progression.
*   **GameOverController**: Manages game state transitions and UI feedback.

## 🚀 Getting Started

1.  Clone the repository.
2.  Open the project in **Unity 2022.3** (or later).
3.  Load the `MainScreen` scene from `Assets/Scenes`.
4.  Press **Play** to start running!

## 🎮 Controls

*   **Swipe Left/Right**: Move between lanes.
*   **Swipe Up**: Jump (if implemented).
*   **Swipe Down**: Slide (if implemented).

## 📁 Project Structure

*   `Assets/Script`: Core C# game logic.
*   `Assets/Prefab`: Reusable game objects for levels and items.
*   `Assets/Scenes`: Main menu and gameplay scenes.
*   `Assets/Audio`: Sound assets.
*   `Assets/TextMesh Pro`: UI styling and fonts.
